This is the website for the Dartmouth Student Alliance for Ukraine. 
Frontend Deploy Link: https://dsau-website.onrender.com/

Tech Stack:
  Frontend     
  - React 19 with TypeScript                                                                                           
  - Vite + vite-react-ssg (static site generation)
  - React Router for routing                      
  - Tailwind CSS v4 for styling                                                                                          
  - @portabletext/react for rendering rich text
                                                                                                                         
  CMS / Backend                                                                                                          
  - Sanity v3 (headless CMS)
  - Content hosted on Sanity Cloud                                                                                       
  - Studio hosted at dsau.sanity.studio    
                                                                                                                         
  Fonts                                                                                                                  
  - National2 (custom)
  - DartmouthRuzicka (custom)                                                                                            
                                           
  Deployment                                                                                                             
  - Frontend: Render.com (static site)
  - Studio: Sanity's own hosting                                                                                         
  - Repo: GitHub
